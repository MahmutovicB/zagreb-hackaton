# Zagreb Hackathon — *Gdje Živjeti*

We are building a software solution for the **Cursor hackathon in Zagreb**, on the topic **“Build something Zagreb wants.”**

## About the project

**Gdje Živjeti** is a Zagreb-focused web app idea: an **AI-assisted neighborhood matchmaker** plus a **live communal-works radar**. Someone hunting for an apartment describes what they need in plain language (for example family, no car, quiet, commute to a part of town) and gets **ranked neighborhoods**, a **map view**, and signals drawn from **official Zagreb open data** ([data.zagreb.hr](https://data.zagreb.hr))—things like active road works, kindergarten capacity, air quality, transit, and links toward local listings.

**Planned pillars** (see [PLAN.md](PLAN.md) for full spec):

- **Neighborhood matchmaker** — natural language → structured criteria → scored districts with explanations and exploration UI.
- **Komunalni radar** — overlay of **Plan komunalnih aktivnosti** and related city layers so users see construction, closures, and maintenance in context.

Implementation details, APIs, and file layout are documented in **PLAN.md**.
